## Install
 In the root of the project run
 ```
 npm install
 bower install
 ```
 If you dont have grunt install it
 ```
 npm install grunt -g
 npm install grunt-cli -g
 ```
 ### Run the project
 In the root of the project run
 ```
 grunt serve
 ```
 This will run the app in the port 9000, open a tab in your browser and reload it upon any change
